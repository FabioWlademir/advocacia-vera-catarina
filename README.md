### 🚀 Destaque do Portfólio

#### [Advocacia Vera Catarina](https://veracatarina.github.io/)
Portal jurídico de alta performance desenvolvido para o escritório Advocacia Vera Catarina, localizado em Sapucaia do Sul.

* **O Desafio:** Criar um site estático rápido, seguro e otimizado para conversão de leads via WhatsApp.
* **Solução Técnica:**
  * **Performance:** Estrutura estática com carregamento instantâneo.
  * **UX/UI:** Design responsivo "Mobile-First" focado em acessibilidade imediata.
  * **SEO:** Otimização estratégica para busca local em Sapucaia do Sul (Direito Criminal, Digital e Trabalhista).
  * **Conversão:** Implementação de gatilhos de WhatsApp para atendimento 24h.
